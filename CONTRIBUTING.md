# Contributing to SeismoWatch

Thank you for your interest in contributing! 🎉

## How to Contribute

1. **Fork** this repository
2. **Edit** `index.html` directly — no build process needed
3. **Test** by opening `index.html` in your browser
4. **Submit** a Pull Request with a clear description

## Areas We'd Love Help With

- 🗺 New data sources (TMD Thailand, JMA Japan, GeoNet NZ)
- 🌐 UI language toggle (English / Thai / Japanese)
- 📱 Mobile UX improvements
- 🎨 Additional map tile themes
- 🔧 Reliable CORS proxy alternatives
- 🐛 Bug fixes and performance improvements

## Guidelines

- Keep it **single-file** — all changes go in `index.html`
- No external dependencies beyond the existing CDN libraries
- Test on both desktop and mobile before submitting
- Add a comment in the code explaining what you changed and why

## Reporting Bugs

Open an [Issue](../../issues) and include:
- Browser and OS
- Steps to reproduce
- Screenshot if possible
